# Flutter Shop

> Vitrine e catálogo de produtos em Flutter

[![Licença: MIT](https://img.shields.io/badge/Licen%C3%A7a-MIT-blue.svg)](./LICENSE) ![Grátis](https://img.shields.io/badge/pre%C3%A7o-Gr%C3%A1tis-brightgreen)

Loja/marketplace completa em Flutter 3.44 + Material 3, com 9 telas e fluxo de compra ponta a ponta: vitrine com busca, categorias e grid de produtos; página de produto com seletor de tamanho/cor e barra fixa de adicionar; carrinho com cupom e resumo; checkout com endereço e pagamento; confirmação; categorias; busca com filtros; favoritos; e perfil com histórico de pedidos. Tema claro/escuro, arquitetura em camadas com MVVM, go_router e providers.

## 🧱 Stack

- Flutter 3.44
- Dart 3
- Material 3
- go_router
- provider
- http

## ✨ Recursos

- Flutter 3.44 (último stable) via FVM
- Material 3 com tema claro e escuro
- Arquitetura em camadas (data/domain/ui) + MVVM
- Navegação declarativa com go_router
- Modelos com JSON (fromJson/toJson) + serviço http
- Catálogo em grid, busca, categorias e favoritos
- Layout responsivo (mobile, tablet, desktop)
- Android · iOS · Web · Windows + testes de widget

## 🚀 Como rodar

Requer o [Flutter SDK](https://docs.flutter.dev/get-started/install) instalado (canal stable).

```bash
flutter pub get
flutter run
```

Para rodar na web: `flutter run -d chrome`. Para gerar o APK: `flutter build apk`.
